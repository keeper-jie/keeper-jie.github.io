红黑树

规则：

  1. 每个节点颜色都为红色或黑色
  2. 树的根始终是黑色的
  3. 没有两个相邻的红色节点（每个红色节点都有两个黑色子节点）
  4. 从节点到任何叶子节点的路径中都有相同数量的黑色节点
    5.所有的叶子都是黑色

和之前数据结构中学到的平衡树AVL树很像，都有插入，调整操作。
有左左，左右，右右，右左四种情况。

STL

STL组成 含义
容器 封装数据的模板类
算法 数据结构算法都被设计成一个个的模板类
迭代器 对容器中数据进行读写操作
函数对象 如果一个类将()运算符重载为成员函数，那么这个类为函数对象类，类的对象就是函数对象（仿函数）
适配器 使一个类的接口适配成用户指定的形式，使两个类在一起工作
内存分配器 为容器类模板提供自定义内存申请和释放功能

STL头文件（<iterator> <functional> <vector> <deque> <list> <queue> <stack> <set> <map> <algorithm> <numeric> <memory> <utility>）

STL容器

序列容器

主要包括vector，list，deque，元素无序

排序容器

主要包括set，multiset，map，multimap，元素有序

哈希容器

主要包括unordered_set,unordered_multiset,unordered_map,unordered_multimap，元素无序，位置由哈希函数确定

STL迭代器

输入迭代器

输出迭代器

前向迭代器（forward iterator）：支持p++

双向迭代器（bidirectional iterator）：可以进行p–操作

随机访问迭代器（random access iterator）：

  • p+=i：p后移i个元素
  • p-=i：p前移i个元素
  • p+i：返回p后面第i个元素的迭代器
  • p-i：返回p前面第i个元素的迭代器
  • p[i]：返回p后面第i个元素的引用

C++中的数组是一个容器，指针为迭代器，而且是随机访问迭代器。
例如：对于数组int a[10],int* 类型的指针为迭代器，a+1,a+2等均为迭代器

序列容器

array<T,N>:数组容器，长度固定不变
vector<T>:长度可变，在尾部增加删除元素时间复杂度O(1)，其他位置O(n)
deque<T>:长度可变，首尾都可以插入数据，时间复杂度O(1)，某一位置插入删除O(n)
list<T>:双向链表，插入删除数据时间复杂度O(1)，访问数据慢，要从第一个节点/最后一个节点开始沿着链表访问
forward_list<T>:单链表形式组织元素，只能从第一个元素开始访问，比list快，更省内存

排序容器(存储Key-value,采用红黑树实现)

排序容器 特点
map 各个元素的键唯一，默认升序排序
set 元素键和值相同，默认升序
multimap 与map区别：键可以重复
multiset 和set区别：元素值可以重复

哈希容器

无序容器 功能
unordered_map 存储键值对，键值对值不允许重复，键值对无序
unordered_multimap 与unordered_map区别，允许多个键相同的键值对
unordered_set 直接存储数据元素本身，无元素元素
unordered_multiset 和unordered_set区别：允许值重复元素

和排序容器区别，在查找指定键平均时间复杂度O(1),如果使用迭代器遍历容器中元素，则其执行效率不如排序容器

C++（STL）容器适配器

stack<T>:后入先出
queue<T>:先入先出
priority_queue<T>:保证最大元素在最前面的队列，封装了vector容器的适配器类模板

容器适配器 基础容器筛选条件 默认使用的基础容器
stack 包含以下成员函数：empty() size() back() push_back() pop_back(),基础容器：vector，deque，list deque
queue 成员函数:empty() size() front() back() push_back() pop_front(),满足条件的基础容器：deque,list deque
priority_queue c成员函数：empty() size() front() push_back() pop_back(),基础容器：vector，deque vector

C++ STL迭代器适配器

名称 功能
reverse_iterator 重新定义递增（++）和递减（–）运算符，实现对容器的逆序遍历
inserter或者inserter_iterator 在容器任意位置添加新元素,不能用于个数固定的容器
流迭代器（istream_iterator/ostream_iterator）流缓冲迭代器（istreambuf_iterator/ostreambuf_iterator) 输入流从文件或者键盘读取数据；输出流迭代器将数据输出到文件或者屏幕上。输入流缓冲区迭代器从输入缓冲区逐个读取数据，输出流缓冲区迭代器将数据逐个写入输出流缓冲区
